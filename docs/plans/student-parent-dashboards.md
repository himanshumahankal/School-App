# Student & Parent Dashboards

**Status:** 📋 Planned

## Overview
Planned dashboards for Students and Parents to track academic progress and communication.

## Proposed Features
- **Student Dashboard:**
  - View current class schedule.
  - View upcoming exams and assignments.
  - Download study materials uploaded by teachers.
  - View personal attendance records.
- **Parent Dashboard:**
  - View children's academic performance.
  - Track children's attendance.
  - View and track fee payment history.
  - Receive announcements and alerts.

## Proposed Implementation Details
- **Controllers:** `App\Http\Controllers\Student\StudentController`, `App\Http\Controllers\Parent\ParentController`
- **Frontend Pages:** 
  - `resources/js/pages/student/dashboard.tsx`
  - `resources/js/pages/parent/dashboard.tsx`
